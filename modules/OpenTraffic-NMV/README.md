# OpenTraffic-NMV

Public delivery repository for non-motor vehicle detection delivery bundle.

## Included Content

1. Linux shared library in `artifacts/`
2. example outputs in `examples/`
3. validation summaries in `validation/`
4. public manifest in `manifests/public_manifest.json`

## Quick Start

1. Review `manifests/public_manifest.json`.
2. Review `validation/public_validation_summary.json`.
3. Integrate the `.so` from `artifacts/`.
4. Compare runtime outputs with the example files in `examples/`.
